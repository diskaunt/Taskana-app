# Инструкция по работе с проектом

## Установка зависимостей

Перед началом работы необходимо установить все зависимости проекта. Выполните команду в терминале:

```bash
npm install
```

## Запуск проекта в режиме разработки

Для запуска проекта в режиме разработки используйте команду:

```bash
npm run dev
```

## Сборка проекта (production)

Для сборки проекта в production-версию выполните:

```bash
npm run build
```
